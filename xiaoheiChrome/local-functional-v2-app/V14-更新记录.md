# 小黑多开器 V14.0.0

## 本次完成

- 增加逐环境一致性检查：Profile 隔离、数据保留、代理出口、时区、地理位置、WebRTC、UA、操作系统标签、图形接口和媒体权限。
- 增加 WebGPU“真实 / 禁用”隐私控制，并与现有 WebGL、Canvas 设置并列展示。
- 合并 Chrome 的 `--disable-features` 参数，避免代理配置覆盖 WebGPU 禁用设置。
- 环境概要明确区分真实值、标准兼容性模拟和禁用策略。
- 更新应用版本为 `14.0.0`。

## 已有且继续保留

- 每个环境使用独立 Chrome 用户目录。
- Cookie、LocalStorage、IndexedDB、历史记录、书签和缓存按 Profile 隔离。
- HTTP / HTTPS / SOCKS5 代理配置、认证桥接及出口 IP 检测。
- WebRTC 非代理 UDP 限制、网站定位权限和媒体设备权限。

## 明确不包含

- 不提供规避平台风控或反机器人检测的功能与保证。
- 不伪造 Canvas、WebGL、WebGPU、字体、CPU、GPU 或硬件标识。
- 不包含 CloakBrowser 专有浏览器二进制，也不复制其受限内核代码。
- macOS、Linux、Android、iOS 标签不等于真实操作系统或真实设备。

## CloakBrowser 借鉴边界

仅参考其公开包装器中的 Profile 生命周期、配置校验和版本状态设计。CloakBrowser 的定制 Chromium 二进制受单独许可证约束，不随本项目分发。
